# #29 JSON Key Picker

A minimalist developer tool to extract specific top-level keys from JSON data. No more manual filtering—just paste, pick, and copy.

## Features
- **1 Tool / 1 Action**: Focuses solely on key extraction.
- **Auto-Analysis**: Immediately identifies available keys upon pasting.
- **Live Preview**: Real-time JSON generation as you select keys.
- **Mobile Optimized**: High-contrast UI with forced iOS Safari visibility fixes.
- **Privacy First**: Browser-only execution. No tracking, no ads, no login.

## Usage
1. Paste your JSON into the "Paste JSON Here" area.
2. Select the keys you want to keep from the generated chips.
3. Click "Copy Result" to get your filtered JSON.
